python 的字符串格式化讲解python 字符串格式化有很多方式，很有必要选一种实用的。 用 f 或者 F 进行格式化，直接加载字符串前面，大括号内的代码会以表达式的方式运行。 这种方式出现在 python3.6 版本，在版本允许的情况下，这种方式更简洁、运行也更快。 123age = 1F'age is {age}'f'age is {age}' 划重点：f 的格式化方式中可以用 函数。 1234def test(val): return val+1 f'age is {test(age)}' 传统方式 % 最老的一种格式化方式，缺点是在变量比较多的时候，很容易看花眼。 str.format 方式 这种是为了优化 % 的方式，python2.6的时候提出来的，如果版本比较低，用这个也行。
